POLITIQUE DE CONFIDENTIALITÉ

Dernière mise à jour : 30 avril 2026

Ce texte est prêt à l'usage. Une relecture juridique finale reste recommandée avant diffusion publique, surtout parce que l'application manipule des données pouvant relever de la santé.

1. Responsable du traitement

- Nom / raison sociale : Projet Glucose For Watch
- Adresse : Communication sur demande via e-mail vie privée
- E-mail vie privée : privacy.gfw@proton.me

2. Données traitées

Selon la configuration, Glucose For Watch peut traiter :

- identifiants Dexcom Share saisis par l'utilisateur ;
- paramètres de configuration ;
- valeurs de glycémie affichées ou synchronisées ;
- état de synchronisation ;
- informations techniques sur le téléphone, la montre et la connectivité ;
- journaux techniques limités au fonctionnement.

3. Données de santé

Les valeurs de glycémie peuvent être considérées comme des données de santé.

4. Finalités

Les données sont utilisées pour :

- connecter l'utilisateur à Dexcom Share ou une source compatible ;
- récupérer et afficher une glycémie ;
- transmettre la valeur à une montre Wear OS ;
- maintenir la sync active ;
- diagnostiquer les erreurs techniques ;
- améliorer la fiabilité locale de l'application.

5. Destinataires

Les données sont destinées à l'utilisateur, à l'application sur ses appareils, aux services tiers explicitement configurés et aux prestataires techniques réels s'il en existe.

Prestataires techniques : Dexcom Share (Dexcom, Inc.), Google Play Services (Google LLC), GitHub (hébergement code source)

6. Stockage

Certaines données peuvent être stockées localement pour faire fonctionner l'application : configuration Dexcom, dernier état de sync, dernière valeur connue et préférence de montre principale.

Les secrets doivent rester sur le téléphone et ne pas être stockés dans le module Wear.

7. Conservation

Les données sont conservées le temps nécessaire au fonctionnement de l'application, sauf suppression par l'utilisateur, désinstallation ou obligation légale contraire.

8. Sécurité

L'application doit limiter les logs sensibles, protéger le stockage local et éviter d'exposer les identifiants Dexcom.

9. Droits

Selon le droit applicable, l'utilisateur peut disposer de droits sur ses données.

Modalités de contact : Demande par e-mail à privacy.gfw@proton.me

10. Autorité de contrôle

Autorité compétente : CNIL (France)
