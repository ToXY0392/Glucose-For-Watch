# Mode d'emploi

## Installation

- Téléphone : installer `mobile-debug.apk`
- Montre : installer `wear-debug.apk`

## Première configuration

1. Ouvrir Widget G7 sur téléphone.
2. Accepter les textes légaux.
3. Se connecter à Dexcom Share.
4. Vérifier que la montre est connectée.

## Utilisation

- Vérifier la valeur dans la tuile.
- Utiliser l'action de refresh en cas de doute.
- Garder l'app autorisée en arrière-plan pour une sync stable.

## Dépannage court

- Valeur figée : retirer/remettre la complication.
- Pas de sync : relancer l'app téléphone puis tester la sync.
