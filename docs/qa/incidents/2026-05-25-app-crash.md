## Cause racine (logcat — X.4)

```
FATAL EXCEPTION: main
Process: com.widgetg7.mobile
java.lang.RuntimeException: Unable to create service
  com.widgetg7.mobile.sync.ActiveGlucoseSyncService
Caused by: android.app.ForegroundServiceStartNotAllowedException:
  Time limit already exhausted for foreground service type dataSync
  at android.app.Service.startForeground(Service.java:775)
  at com.widgetg7.mobile.sync.ActiveGlucoseSyncService.onCreate(ActiveGlucoseSyncService.kt:32)
```

**Explication :** Android 14+ limite le temps cumulé des services foreground `dataSync` lancés depuis l'arrière-plan. Quand la quota est épuisée, `startForeground()` lève une exception **non gérée** dans `onCreate()` → crash fatal (« a cessé de fonctionner »).

**Déclencheurs possibles dans le code :**
- `ActiveGlucoseSyncController.start()` depuis `MainActivity`, `PhoneBootReceiver`, `PhoneAutoSyncReceiver`
- `PhoneWearRefreshRequestService` → `syncNow()` → redémarrage service
- Schedulers redondants (alarm + WorkManager + service) qui relancent le FGS
