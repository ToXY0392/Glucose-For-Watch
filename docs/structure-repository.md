# Structure du dépôt

## Dossiers principaux

- `mobile/` : app téléphone
- `wear/` : app montre
- `core/` : modèles et contrats
- `feature/` : logique métier modulaire
- `docs/` : documentation
- `scripts/` : scripts utilitaires

## Sorties build

- APK mobile : `mobile/build/outputs/apk/debug/`
- APK wear : `wear/build/outputs/apk/debug/`
