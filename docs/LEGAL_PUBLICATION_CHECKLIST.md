# Checklist juridique avant diffusion

Date : 30 avril 2026

Les documents juridiques existent, mais ils ne sont pas prets pour une diffusion publique.

## A completer dans les CGU

- nom ou raison sociale de l'editeur ;
- forme juridique ;
- adresse ;
- e-mail de contact ;
- responsable de publication ;
- droit applicable ;
- juridictions competentes ;
- contact final.

Document : [CGU.md](CGU.md)

## A completer dans la confidentialite

- identite du responsable de traitement ;
- adresse ;
- e-mail vie privee ;
- prestataires techniques reels, ou mention explicite qu'il n'y en a pas ;
- autorite de controle applicable.

Document : [POLITIQUE_CONFIDENTIALITE.md](POLITIQUE_CONFIDENTIALITE.md)

## A reporter dans l'app

Les memes informations doivent etre reportees dans les textes embarques :

- [cgu.txt](../mobile/src/main/res/raw/cgu.txt)
- [politique_confidentialite.txt](../mobile/src/main/res/raw/politique_confidentialite.txt)

## Decision

Ne pas publier l'APK hors test prive tant que cette checklist n'est pas terminee et relue.
