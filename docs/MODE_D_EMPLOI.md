# Mode d'emploi

Ce mode d'emploi reprend le parcours utile de Widget G7.

## 1. Installer

- Installez l'APK mobile sur le telephone.
- Installez l'APK Wear sur la montre.
- Ouvrez l'application mobile.

## 2. Connecter Dexcom

- Ouvrez `Dexcom`.
- Acceptez les textes requis.
- Saisissez vos identifiants Dexcom Share.
- Choisissez la region `Europe` ou `US`.
- Enregistrez.

Apres une connexion reussie, l'application tente aussi d'envoyer la derniere glycemie vers la montre.

## 3. Configurer la montre

- Ouvrez `Parametres`, puis `Parametres montre`.
- Verifiez que la montre est detectee.
- Lancez `Tester la synchronisation`.
- Ajoutez ensuite la tile glucose ou la complication depuis Wear OS.

## 4. Actualiser

- Depuis l'accueil telephone, utilisez le bouton `Sync`.
- Depuis la montre, utilisez le bouton d'actualisation de la tile.

L'accueil affiche maintenant un diagnostic court : Dexcom a configurer, derniere sync, ou derniere erreur.
