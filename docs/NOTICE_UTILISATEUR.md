# Notice Utilisateur

## 1. Connecter Dexcom

- Ouvrir `Dexcom`
- Appuyer sur `Se connecter`
- Entrer vos identifiants
- Appuyer sur `Enregistrer`

## 2. Vérifier la montre

- Revenir à l'accueil
- Vérifier que la montre est `Connectée`

## 3. Sur la montre

- Ajouter le `Glucose Tile`
- ou ajouter la complication `Glucose`

## Si besoin

- Utiliser le refresh
