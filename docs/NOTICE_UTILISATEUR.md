## 🟢 Démarrage Rapide

```text
╭─ Première configuration ───────────────╮
│  1. Installer l'app mobile             │
│  2. Installer l'app Wear               │
│  3. Ouvrir Widget G7 sur le téléphone  │
│  4. Accepter les textes requis         │
│  5. Se connecter à Dexcom Share        │
│  6. Tester l'envoi vers la montre      │
│  7. Ajouter la tile ou complication    │
╰────────────────────────────────────────╯
```

---

## 🔁 Utilisation

| Action | Effet |
| --- | --- |
| `Synchroniser` | Relance une actualisation depuis le téléphone |
| Refresh montre | Demande au téléphone de relancer la sync |
| Tile Wear OS | Affiche la dernière valeur reçue |
| Complication | Affiche la valeur sur un cadran compatible |

```text
Dexcom Share -> téléphone -> Wear OS
```

La montre ne se connecte pas directement au capteur G7 dans cette version.

---

## 🧪 En Cas De Problème

Vérifier :

| Point | À contrôler |
| --- | --- |
| 📶 Réseau | Le téléphone a internet |
| ⌚ Wear OS | La montre est connectée |
| 🔐 Dexcom | Le compte Dexcom Share est toujours configuré |
| 🔋 Batterie | Widget G7 est autorisé en veille |
| 🧱 Surface | Tile ou complication ajoutée sur la montre |

---

## ⚠️ Important

> Widget G7 n'est pas un dispositif médical officiel.
>
> En cas de doute, vérifier la valeur dans l'application Dexcom G7, le récepteur Dexcom ou une solution officielle adaptée.

Le mode direct `capteur G7 -> Wear OS` est seulement une piste expérimentale.
