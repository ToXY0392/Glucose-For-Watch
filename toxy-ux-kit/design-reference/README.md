# Design reference (replaces optional Figma)

> **Figma is not required.** This folder is the visual handoff for developers and reviewers.

## View

Open in any browser:

```
toxy-ux-kit/design-reference/index.html
```

Or regenerate from tokens:

```powershell
py -3 toxy-ux-kit/tools/export-design-reference.py
```

## Contents

- ToXY chrome + AGP medical swatches (from JSON tokens)
- Component previews: wear tile, wear status, mobile hero, sync pills
- Links to PNG references in `assets/references/`

## Figma (optional)

If you prefer Figma for collaborative design, see [`../figma/FIGMA-HANDOFF.md`](../figma/FIGMA-HANDOFF.md).  
The repo source of truth remains `tokens/` + this HTML reference.
